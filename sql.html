Sintaxe para consultar a quantidade de clientes por estado e por fonte de tráfego em uma tabela de veículos de comunicação.

            select 
            STATE,
            SOURCE,
            count(*) as Qnt 
            from PEOPLE
            where id <= 100
            group by STATE, SOURCE
            order by STATE, SOURCE
